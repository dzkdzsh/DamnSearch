# DamnSearch 🔍

**DamnSearch** 是一个专为苦命英语人/考研党打造的“英语单选题题目溯源搜题神器”。它通过打包成独立的桌面客户端，让你可以把几百道题一键查出来源，从而帮助你分辨老师出的是“早年老四六级题库”、“职称英语题”、“成人学位英语考题” 还是最新雅思托福。

## ✨ 特性说明 
- 🚀 **开箱即用**: 不需要安装库资源库和环境（附带独立 `.exe`）.
- 🎨 **图形化界面 GUI**: 将一列题目复制丢进去，点“无情搜题”傻傻等结果出来即可。它会自动剔除前面的 `ABCD` 盲注项和无用换行。
- ⚡ **无锁死并发搜题**: 依靠内置多线程引擎，界面永远不会卡死冰冻，后台检索速度取决于你的手速与你的网络。

## 🎯 适合哪些人使用
1. 那些面临老师“海量刷老题任务”却找不到答案出处的英语学习者
2. 没有计算机专业基础的纯“小白”同学们 （直接双击 DamnSearch.exe 即可食用！）

## 🚀 如何生成它 / 本地开发
如果你想要自行使用 `python` 运行或二次开发它，需要有 Python 的环境：
1. 确保你安装了 Python (3.x 版)
2. 克隆本仓库到你的系统本地盘下   `git clone ...`
3. 运行代码： `python damn_search.py` 

## 📦 如何打包 (一键导出 .exe)
如果你想把程序发送给你的朋友当礼物送人：
安装 `pyinstaller` 并通过指令将其构建成一个没有黑框框后羿执行的单文件免安装版本（Standalone Tool）：
```powershell
pip install pyinstaller
pyinstaller -w -F damn_search.py --name "DamnSearch"
```
打包结束后，会在 `dist/` 文件夹下生成一个 `DamnSearch.exe`，直接发给同学即可！
